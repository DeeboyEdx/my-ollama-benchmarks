## About

This site is auto-generated by [`Publish-Website.ps1`](https://github.com/DeeboyEdx/z440-ollama-benchmarking/blob/master/Publish-Website.ps1) from the [z440-ollama-benchmarking](https://github.com/DeeboyEdx/z440-ollama-benchmarking) repository.

## Features

- Sortable, filterable model comparison table
- VRAM feasibility indicators (observed RAM from `ollama ps`)
- Interactive charts: TTO by task, generation speed, RAM usage vs VRAM limit
- Quality grade heatmap with per-response tooltips
- Metric glossary and quality scoring reference

## Updating

Run `Publish-Website.ps1 -Push` from the main benchmarking repo to regenerate `data.js` and push the latest results here.
